# glaz-boga
Глаз Бога телеграм бот, самый мощный OSINT-бот для пробива информации и поиска людей. Ищет по номеру телефона, нику, email и соцсетям, показывает оператора и возможные привязки, сверяет базы утечек и открытые источники. Быстрый запуск в Telegram, удобный интерфейс, анонимность, точные результаты для проверки личности и безопасности.
